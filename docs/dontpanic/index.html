libdontpanic

libdontpanic was created to test Alien::Base. Its purpose was inspired by the Hitchhiker's Guide to the Galaxy by Douglas Adams. it povides a single function dontpanic which returns the value of 42. This being the answer to Life, the Universe and Everything.
