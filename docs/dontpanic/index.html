libdontpanic

libdontpanic was created to test Alien::Base. Its name was inspired by The Hitchhiker's Guide to the Galaxy by Douglas Adams. It was originall written by Joel Berger, and is currently maintained by Graham Ollis.

It provides a single function dontpanic which returns the value of 42, the answer to Life the Universe and Everything.
